HRMS Application Guide

How to use the Human Resource Management System

User Roles

Admin

  • Create and manage all employees
  • View employees across all departments
  • Approve leave requests
  • Generate payroll for any employee
  • View all attendance records
  • Advance the system date

Manager

  • View employees in their department only
  • Approve leave requests
  • Generate payroll for department employees
  • View attendance records
  • Advance the system date

Employee

  • Mark daily attendance
  • Submit leave requests
  • View their own leave history
Dashboard Guide

Admin Dashboard

Employees Tab
Create new employees and view all employees across departments
Leaves Tab
View all leave requests and approve pending ones
Attendance Tab
Enter an employee ID to view their attendance records
Payroll Tab
Generate monthly payroll and view all payroll records

Manager Dashboard

My Department Tab
View employees in your department only
Leaves Tab
View and approve pending leave requests
Attendance Tab
Enter an employee ID to view their attendance
Payroll Tab
Generate and view payroll for your department

Employee Dashboard

Mark Attendance
Enter hours worked today and submit. Hours over 8 count as overtime at 1.5x rate
Request Leave
Select start date, end date, and type (Paid or Unpaid)
My Leave Requests
View all submitted leave requests and their approval status

Public

3
Create an Employee — Employees Tab
Admin
In the Employees tab, fill in all fields to create a new employee and their linked user account in one step.
Full Name: John Doe Employee Number: EMP001 Username: johndoe Password: john123 Department: Human Resources Hourly Rate: 50
The employee record and user account are created together automatically.
4
Register a Manager account
Public
Go to the Register page and create a Manager user in the same department.
Username: manager1 Password: manager123 Role: Manager Department: Human Resources
5
Login as Employee and mark attendance
Employee
Logout of the admin account. Login as johndoe. On the Employee Dashboard, enter hours worked and click Mark Attendance.
Hours Worked: 9 (8 regular + 1 overtime)
You can only mark attendance once per system date. Use Next Day to advance the date.
6
Advance the system date
System
Click the Next Day button in the navbar to advance the system clock. Repeat steps 5 and 6 a few times to build up attendance data.
7
Request leave as Employee
Employee
Still logged in as johndoe, submit a leave request.
Start Date: 2026-04-01 End Date: 2026-04-03 Type: Paid
8
Approve leave as Admin or Manager
Admin
Login as admin1 or manager1. Go to the Leaves tab and click Approve on the pending request.
9
Generate Payroll
Admin
Go to the Payroll tab. Enter the employee ID and select the month to generate payroll. Gross pay is calculated from attendance records and approved paid leave.
Employee ID: 1 Month: 2026-03
Payroll can only be generated once per employee per month.
10
Test Manager Dashboard
Manager
Login as manager1. Notice the Manager Dashboard only shows employees from their department and has a department label in the header.
