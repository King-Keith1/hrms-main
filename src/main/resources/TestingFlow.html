Payroll Strategy Pattern — Contract Types

Full Time

Regular = hourlyRate x hours Overtime = hourlyRate x OT x 1.5 Leave = hourlyRate x 8 x leaveDays Gross = regular + overtime + leave
10hrs x R50 = R550/day (includes R150 overtime)

Part Time

Gross = hourlyRate x hoursWorked (no overtime, no paid leave)
10hrs x R50 = R500/day (no overtime bonus)

Contract

Daily Rate = hourlyRate x 8 Gross = dailyRate x daysWorked (flat rate regardless of hours)
R50 x 8 = R400/day flat rate
User Roles

Admin

  • Create and manage all employees
  • View employees across all departments
  • Approve leave requests
  • Generate payroll for any employee
  • View all attendance records
  • Advance the system date

Manager

  • View employees in their department only
  • Approve leave requests
  • Generate payroll for department employees
  • View attendance records
  • Advance the system date

Employee

  • Mark daily attendance
  • Submit leave requests
  • View their own leave history

Employee

Hours Worked: 10 (8 regular + 2 overtime)
Hours above 8 are automatically recorded as overtime and paid at 1.5x for Full Time employees.

Advance the System Date
System
Login as Admin or Manager and click the Next Day button in the navbar. The current system date updates instantly. Repeat steps 4 and 5 a few times to build up attendance data.
